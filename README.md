# Odor-Monitoring-Datasets-2025/26
Este repositório contém os datasets obtidos pela a estação de monitoramento de odor nos experimentos realizados nos anos de 2025 e 2026.

### Estrutura padrão dos arquivos para cada experimentos
```text
.
└── nome-do-experimento/
    ├── README.md
    ├── data/
    │   └── dados-raw.csv
    ├── notebook/
    │   └── caderno-de-analise.ipynb
    └── output/
        ├── plot/
        │   └── grafico-plotado.png
        └── processed/
            └── dataset-processado.csv
```
